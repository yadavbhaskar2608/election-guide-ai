# 🗳️ Election Guide AI Assistant (India)

A simple, clean, and beginner-friendly web application that helps Indian citizens understand the election process, voting steps, timelines, and their democratic rights.

## ✨ Features

### 🏠 Home Section
- Hero banner with animated background blobs in India's tricolour (saffron, green, navy)
- Live stats: 96 Cr+ registered voters, 543 Lok Sabha seats, 7 election phases
- Quick navigation buttons to all sections

### 🤖 AI Chatbot — "Chunav Mitra"
- Conversational chatbot with predefined answers in beginner-friendly language
- Answers questions on:
  - How to vote in India (step-by-step)
  - Voter ID / EPIC card – how to apply and check
  - NOTA – what it is and how to use it
  - Documents accepted at the polling booth
  - Voting rules – Dos and Don'ts
  - EVM & VVPAT explained
  - Voter eligibility
- **Quick reply buttons**: How to vote · Check Voter ID · Voting Rules · What is NOTA · Documents needed
- Typing animation, message timestamps, clear chat button

### 📋 Voting Guide
- 4 step-by-step cards:
  1. Register as a Voter
  2. Check Name in Voter List
  3. Visit Polling Booth
  4. Vote Using EVM
- Tips and links to official ECI resources
- Banner listing all 12 accepted photo ID documents

### 📅 Election Timeline
- 4-phase vertical timeline:
  1. Election Announcement & MCC
  2. Nomination & Campaign Period
  3. Voting Day (Poll Day)
  4. Vote Counting & Result Day
- Did You Know? facts about Indian elections

## 🚀 Getting Started

No installation required! Just open `index.html` in any browser.

```bash
git clone https://github.com/yourusername/election-guide-ai.git
cd election-guide-ai
# Open index.html in your browser
```

Or deploy directly to **GitHub Pages** (see below).

## 🌐 Deploy to GitHub Pages

1. Push this repo to GitHub (see below)
2. Go to **Settings → Pages**
3. Set **Source** to `main` branch, `/ (root)` folder
4. Click **Save** — your site will be live at `https://yourusername.github.io/election-guide-ai`
